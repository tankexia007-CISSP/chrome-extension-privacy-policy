Privacy Policy for Chrome Browser FullPage Screenshot Extension

Last Updated: 2026-04-22

1. Information Collection

This extension does NOT collect, store, or transmit any personal user information, browsing history, account data, identity credentials or sensitive privacy content to external servers.

All user configuration settings are stored locally only inside your Chrome browser.

2. Data Storage Method

All extension data stays on your local device permanently. There is no cloud backup, no remote upload, and no cross-device data synchronization.

3. Third-Party API Services

This extension does not call any unauthorized third-party APIs, and does not share any user data with external third-party platforms.

4. User Data Deletion Rights

You can completely delete all local saved data of this extension with one click in Chrome extension management page at any time.

5. Browser Permission Detailed Instructions

Storage Permission: Locally store user screenshot format settings, image quality preferences, and paid advanced feature membership authorization status. All data stays inside user browser, never synchronized to cloud servers or shared with third parties.

Downloads Permission: Allow extension to export and download finished full-page screenshot images to user local computer storage. Free users have limited download times, paid subscribed users enjoy unlimited high-quality screenshot download permissions.

Scripting Permission: Inject lightweight rendering scripts into webpage to complete long-page scrolling capture and image stitching. Scripts will not modify original webpage content, track user browsing behavior, or collect any private information.

Identity Permission: Used for Chrome extension account security authentication and browser permission matching. It ensures normal authorization operation of webpage screenshot functions. All identity verification processes are completed locally within Chrome browser. We will never obtain, store or share any user personal identity information, account privacy or sensitive browsing data.

6. Paid Service & In-App Purchase Rules

This extension provides free basic screenshot functions and paid advanced value-added services. Users can purchase official in-app membership to unlock unlimited high-resolution screenshots, no-watermark export and permanent advanced function access.

All payment transactions are processed through official Google Play Billing channels. We will not record, store or obtain user bank card, payment account and financial privacy information.

Paid service rights only apply to the current user browser environment. There is no cross-device account synchronization, and paid fees will not be refunded for normal use of functions.

7. User Rights & Service Restrictions

Free users are subject to quantity limits on webpage screenshot downloads. After successful payment, all restrictions on advanced functions will be lifted immediately.

We strictly prohibit malicious brushing, cracking and tampering with paid authorization status, and reserve the right to restrict abnormal account services.

8. Contact Support

If you have any privacy related questions, please contact the developer support email.
