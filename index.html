Privacy Policy for Chrome Browser Extension

Last Updated: 2026-04-21

1. Information Collection

This extension does NOT collect, store, or transmit any personal user information, browsing history, account data, or sensitive privacy content to external servers.

All user configuration settings are stored locally only inside your Chrome browser.

2. Data Storage Method

All extension data stays on your local device permanently. There is no cloud backup, no remote upload, and no data synchronization.

3. Third-Party API Services

This extension does not call any third-party APIs, and does not share any user data with external third-party platforms.

4. User Data Deletion Rights

You can completely delete all local saved data of this extension with one click in Chrome extension management page at any time.

5. Browser Permission Instructions

All browser permissions applied by the extension are only used to realize normal core functions, and will not be used for any unauthorized data access.

6. Contact Support

If you have any privacy related questions, please contact the developer support email.
